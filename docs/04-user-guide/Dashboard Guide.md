# Dashboard Guide

Dashboard V2 currently includes:

- Platform Overview
- Site Settings
- Navigation
- Legacy Content editor
- Theme editor
- Legacy Settings editor
- Media
- Versions

The platform panels are the future direction. Legacy panels remain during migration.


## Pages Tab

The Pages tab is the first WebsiteSchema-native page management interface.

It allows editing:
- Page title
- Slug
- Type
- Status
- SEO title
- SEO description
- Keywords
- Canonical URL

It also supports:
- Add Page
- Duplicate
- Delete
