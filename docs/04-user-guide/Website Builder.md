# Website Builder

The Website Builder will allow users to create pages from reusable sections.

Future sections:
- Hero
- Text
- Gallery
- Services
- Portfolio
- Testimonials
- FAQ
- CTA
- Contact


## Hero Section

The Hero section is the first universal WebsiteSchema-rendered section.

Supported fields:
- eyebrow
- title
- subtitle
- primaryCtaLabel
- primaryCtaHref
- secondaryCtaLabel
- secondaryCtaHref
- image
- imageAlt
- align
