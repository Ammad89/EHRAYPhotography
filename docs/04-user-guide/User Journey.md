# User Journey Guide

This document describes the full journey of creating a website with Eight Nine Web Studio.

## 1. Sign in

The user opens the dashboard and signs in using an approved CMS account.

## 2. Review platform overview

The user reviews the website status, page count, services, portfolio items and current theme.

## 3. Configure site settings

The user updates:
- Site name
- Business name
- Owner name
- Tagline
- Domain
- Contact details

## 4. Configure navigation

The user updates:
- Primary links
- Service links
- Header CTA
- Visibility
- Sort order

## 5. Build pages

The user creates or edits website pages.

Future page workflow:
- Create page
- Select page type
- Add SEO
- Add sections
- Preview
- Publish

## 6. Manage collections

The user manages reusable content such as services, portfolio, testimonials and FAQs.

## 7. Preview

The user previews the site before publishing.

## 8. Publish

The user publishes the WebsiteSchema. The live website reads the published schema.

## 9. Maintain

The user returns to update content, publish changes, restore versions and add pages over time.
