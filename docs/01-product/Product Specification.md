# Product Specification

This document will become the master feature specification for Eight Nine Web Studio.

## Current Product Areas

- Website creation
- Site settings
- Theme settings
- Navigation
- Footer
- Pages
- Collections
- Media
- Publishing
- Version history
- User permissions
- Multi-site management


## Platform Pages Manager

Status: Started

Capabilities:
- View WebsiteSchema pages
- Edit title
- Edit slug
- Edit page type
- Edit page status
- Edit SEO title
- Edit SEO description
- Edit SEO keywords
- Edit canonical URL
- Duplicate page
- Delete page
- Add new custom page

Current limitation:
Sections are displayed as a count only. Section editing will be added later.


## Platform Sections Manager

Status: Started

Capabilities:
- Select a WebsiteSchema page
- Add a section
- Delete a section
- Duplicate a section
- Move a section up or down
- Change section type
- Change variant
- Change sort order
- Toggle visibility
- Edit raw section data JSON

Current limitation:
Section-specific visual editors and universal rendering are future builds.
