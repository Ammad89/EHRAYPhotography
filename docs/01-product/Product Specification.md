## Platform Pages Manager

Status: Started

Capabilities:
- View WebsiteSchema pages
- Edit title
- Edit slug
- Edit page type
- Edit page status
- Edit SEO title
- Edit SEO description
- Edit SEO keywords
- Edit canonical URL
- Duplicate page
- Delete page
- Add new custom page

Current limitation:
Sections are displayed as a count only. Section editing will be added later.
